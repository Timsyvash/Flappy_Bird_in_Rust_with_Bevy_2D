# Flappy_Bird_in_Rust_with_Bevy_2D  
Я зробив гру Flappy Bird на Rust з Bevy 2D. Тут використовується 2D графіка на Rust.  
## Що таке Rust?  
Rust (укр. Раст, досл. «Іржа») — сучасна багатопарадигмова мова програмування загального призначення. Мова має сильну (строгу) типізацію і зосереджена на безпечній роботі з пам'яттю та забезпеченні високої рівночасності виконання задач (можливість породжувати тисячі й навіть мільйони підпроцесів).  
## Що таке Flappy Bird?  
Flappy Bird — це проста, але надзвичайно складна мобільна аркадна гра, розроблена в'єтнамським програмістом Донгом Нгуєном, де гравець керує пташкою, що летить, змушуючи її пролітати між парами зелених труб, просто натискаючи на екран, щоб пташка злетіла; вона стала вірусною на початку 2014 року, але була видалена розробником через надмірну популярність та залежність, яку вона викликала.  
## Що таке Bevy 2D?  
Bevy 2D – це частина ігрового рушія Bevy, написаного на мові Rust, яка зосереджена на створенні двовимірних (2D) ігор та застосунків, пропонуючи простий, орієнтований на дані фреймворк, що дозволяє розробникам швидко створювати прототипи та повноцінні 2D проекти з акцентом на продуктивність та розширюваність. Він використовує архітектуру ECS (Entity Component System) і має багато прикладів та плагінів для анімації, що робить його популярним вибором серед розробників на Rust.
